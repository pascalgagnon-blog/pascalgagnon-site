Qui je suis ——

J'observe l'économie québécoise depuis l'intérieur.

Je travaille au quotidien avec des entrepreneurs du Lac-Saint-Jean — PME forestières, manufacturiers, détaillants, coopératives agricoles. Depuis mon bureau de directeur de compte chez Desjardins Entreprise, je vois les décisions de financement, les projets d'expansion, les restructurations. Je vois l'économie telle qu'elle fonctionne réellement, pas telle qu'elle est modélisée.

Parallèlement, comme conseiller municipal à Roberval depuis 2021, j'observe comment les décisions politiques locales s'articulent — ou non — avec les réalités économiques régionales. L'écart entre les deux est souvent révélateur.

J'écris pour fixer ces observations. Pas pour informer au sens journalistique — pour analyser, prendre position, forcer la réflexion. La règle d'or de chaque texte : qu'est-ce que le lecteur fera différemment après l'avoir lu?
